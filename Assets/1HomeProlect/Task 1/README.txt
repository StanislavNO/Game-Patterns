	Стратегия:

	1) У игрока есть кнопка действие , если игрок держит в руках 
какойто интерактивный предмет то при нажатии активируеться десйтвие
этого предмета. Возможные предметы(зелье, пульт управления, книга...)

Наследуем все предметы от итрефейса IInteractable с методом Activate()
У игрока есть приватное поле этого интерфейса в которое записывается 
предмет который игрок держит в данный момент.

При нажатии на клавишу действие игрок делает IInteractable.Activate()

	2) У игрока есть снаряжение которое позволяет использовать 
специальную способность этой брони. Разная броня реализует по своему
логику способности.


	Шаблонный метод:

	1) В абстрактном классе Enemy есть метод TakeDamage(int damage). 
В котором от здоровья врага отнимается урон который рассчитывется индивидуально
в классах наследниках. health -= AffectDamage(damage); 

AffectDamage() абстрактный и возвращает int. Наследники реализуют его по своей логике, а родитель
отнимает от здоровьяпересчитанный наследником входящий урон.

	2) В игре есть шахта которая генерит множества разных ресурсов. У нас есть юниты,
у каждого юнита есть свой инвентарь в который помещается только определенный ресурс и свой способ передвижения
есть летающие юниты , плавающие и обычные с ногами :)
В родительском классе определяем алгоритм поведения юнита, а отдельные шаги этой логики реализуем в наследниках.
